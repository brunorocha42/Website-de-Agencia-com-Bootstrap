Projeto académico de um website multi-página para uma agência, desenvolvido com HTML, CSS e Bootstrap.
Este projeto consiste num website completo com várias páginas (Início, Sobre, Portfólio, Ofertas e Contactos). Foi desenvolvido utilizando HTML, CSS e a framework Bootstrap para a estruturação e design.
Aviso de Inspiração: O design e o aspeto visual deste site foram inspirados em websites reais. O objetivo foi recriar esses designs para fins de estudo e treino prático de código.
-Como visualizar: 

  1 Fazer o download do ficheiro ZIP
  2 Extrair a pasta para o computador 
  3 Abrir o ficheiro principal (TrabalhoAvaliacao2.html) num navegador de internet
